# Python × SQLite 接続及びレコード挿入など

- show.py : テーブルを表示

- insert.py : レコード挿入

- delete.py : レコード削除

## 型一覧

| 型名 | 情報 |
|:-----------------|:------------------|
| NULL | NULL値 |
| INTEGER | 符号付整数。1, 2, 3, 4, 6, or 8 バイトで格納 |
| REAL | 浮動小数点数。8バイトで格納 |
| TEXT | テキスト。UTF-8, UTF-16BE or UTF-16-LEのいずれかで格納 |
| BLOB | 入力データをそのまま格納 |
